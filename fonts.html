To Change the font for title at the top of the page, paste over the line shown in red text in the h1 section of Tystyle3.css.


h1 {
font-family: 'Amatic SC', cursive;
font-size: 3em;
color: #ADD8E6;
text-align: center;
}


To Change the font for the names of the videos, paste over the line shown in red text in the h2 section of Tystyle3.css.

h2 {
font-size: 2em;
font-family: 'Amatic SC', cursive;
color: #ADD8E6;
text-align: center;
}
